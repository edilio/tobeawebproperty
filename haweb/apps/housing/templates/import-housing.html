Active Contracts
The structure of the file for Active Contracts should be like:
[tenant_id], first_name, mi, last_name, email, cell_phone, home_phone, work_phone, first_day, last_day, unit_id or address, apartment, city, state, zip_code
New Admissions
The structure of the file for New Admissions should be like:
[tenant_id], first_name, mi, last_name, email, cell_phone, home_phone, work_phone, first_day, last_day, unit_id or address, apartment, city, state, zip_code
Move outs
The structure of the file for Move Outs should be like:
[tenant_id] or [first_name, mi, last_name], unit_id or [address, apartment, city, state, zip_code], move_out_day
